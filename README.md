# TiendaCafePOO

Proyecto desarrollado en Java aplicando Programación Orientada a Objetos (POO).

##  Descripción
Sistema básico de gestión para una tienda de café, donde se aplican conceptos de clases, objetos, encapsulamiento y estructura modular.

##  Tecnologías utilizadas
- Java
- Programación Orientada a Objetos

##  Funcionalidades
- Creación de productos
- Gestión de pedidos
- Uso de clases y métodos
- Aplicación de lógica de negocio

##  Objetivo del proyecto
Fortalecer conocimientos en POO, estructura de proyectos en Java y aplicación de buenas prácticas básicas de programación.
